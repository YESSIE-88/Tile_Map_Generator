# Tile_Map_Generator
Programs to generate a random tile map from given images and then display it

First ensure you have dowloaded the images to use for the tile map into the same directory as the python files

From the terminal type: python Map_File_Generator.py
Follow the instructions to generate the new random tile map

Then run the other python script with: python Map_File_Viewer.py
Enter the name of the tile map you just created and it should be displayed
